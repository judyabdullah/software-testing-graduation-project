# Software Testing Graduation Project

This repository contains a complete software testing project demonstrating hands-on experience in **Manual Testing**, **Automation Testing**, and **API Testing**.

## Project Scope
- Manual test case design and execution
- Automated UI testing
- API testing and validation
- Bug reporting and documentation

## Repository Structure
- **Manual Testing**: Excel-based test cases and test scenarios
- **Automation testing**: Automated test scripts and frameworks
- **API Testing**: API test cases, collections, and responses
- **Bug Reports**: Documented defects with clear reproduction steps

## Tools & Technologies
- Manual Testing techniques
- Selenium / Playwright
- Postman
- Python / Java

## Objective
This project reflects practical testing skills and understanding of the software testing lifecycle.
